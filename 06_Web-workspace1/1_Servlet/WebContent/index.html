서블릿이란?
웹 서비스를 위한 "자바 클래스" 를 말하며
자바를 이용해서 웹을 만들기 위해 필요한 기술이다.
- 사용자의 요청을 받아 처리하고 그에 해당되는 응답페이지를 만들어 다시 사용자에게 전송하는 역할을 하는 자바 클래스 (Controller 의 기능)
- 즉, 웹에서 동적인 페이지를 JAVA 로 구현할수 있게 도와주는 서버측 프로그램 (WAS 서버에서 구동됨)
JAVA 클래스에서 웹 페이지 구현을 위한 HTML 코드가 들어간 구조라고 할 수 있음 (Java 코드로 HTML 구현)
